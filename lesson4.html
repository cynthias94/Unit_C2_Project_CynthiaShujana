Tables

Tables

  • Tables allow web developers to arrange data into a grid like structure
  • Aside from tables with grid border, you can also create tables without border, and position content without it looking like a grid
  • The essential elements for tables are:
  1. <Table> creates the table
  2. <tr> creates a row
  3. <th> creates a heading cell
  4. <td> creates a data cell

Basic Structure of Tables

        <table> border=”1” (border attribute where 1 is the default) 
             <tr> 
               <th>Heading 1</th>
               <th>Heading 2</th>
             </tr>
             <tr>
                <td>Data 1</td>
                <td> Data 2</td>
             </tr>
        </table>

Styling Tables

Table Borders

In HTML 5 we use CSS to create borders. If you want a table with borders, add the following code to the CSS section of your web page on CodePen:
Table, th, td {border:1px solid black;}

Table and Cell Sizes

By default, a table will size itself to fit the contents of the cell within it. But you can force a table (or the cells within it) to have specific sizes. You can specify pixels (px) or percentages of the width or height of the page.

If you want to control the size, write the following into the styling section of CodePen:

          
        Table {
          width:100%;
          height:500px
          }
        th,td{
          width:100px;
          height:50px;
          }

Colspan and Rowspan

Sometimes you want a cell to break access to more than one row or column (or both). In this case, you use the rowspan or colspan attribute.( the number tells how many of either it expands to)

<th colspan=”2”>
<td rowspan=”2”>
