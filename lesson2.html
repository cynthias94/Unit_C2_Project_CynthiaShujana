Comments, Attributes, and More

Comments

Comments are text hidden within an HTML document. In order to create an HTML comment, you must use the characters “<!--” before the comment text and “-->” after it

  • <!-- This is a single line comment
  • <!-- This is a
    multi-line comment -->
  • <!-- You can disable elements like (<h1> Hello World </h1>) by enclosing them in a comment -->

Attributes

Attributes are codes that appear within the opening tag of an element. They are used to modify the element in some way. They general syntax for an attribute is:
<tagName attributeName="value">content</tagName>

Examples of some attributes are:

  • <img src="happyface".jpg>
  • <p style="color:red"> This paragraph will appear in red </p>

Links

Links use the anchor or <a> element to make a hyperlink to another page or to another location on the same page. It is an inline element. To locate the link, you need the href attribute.

  • The general syntax for a link is as follows: <a href="pathToFile">LinkText</a>
  • Syntax for absolute, or external, links can look like: <a href="https://google.com">Google</a>
  • Syntax for relative, or internal, links can look like: <a href="/index.html">Home Page</a>

More Link Attributes

  • Download lets the user download a file
    <a download="fileName">...
  • Target specifies a specific place within the page that the link opens <a href="URL" target="_blank">...

    Some targets:
    _blank opens in a new window/tab
    _top opens at the top of the page
    _self opens within the same iframe
    _parent opens within the parent of the current frame

Embedding Images

Images are embedded on a web page using the <img> element. It is an empty element (no trailing tag) and works inline. To locate the image you must use the src attribute. You must also use the alt attribute to describe the image for screen readers or if the image will not load.
There are two ways to embed images: local images and remote images. A local image is one that is pre-downloaded onto one's device. A remote image is an image that is found on the web through a link.

An example syntax for a local image is as follows:
<img src="happyface.png"alt="Smiley Face">

An example for the syntax of a remote image is as follows:
<img src="https://png.pngtree.com/png-vector/20241014/ourmid/pngtree-playful-smiley-face-clipart-showcasing-a-happy-emoji-suitable-for-use-png-image_14090536.png"alt="Smiley Face">

Sizing Images

You can control the size of images using the width or height attributes, or both. The actual dimensions are specified in pixels. If you only specify solely the height or width, the image will scale proportionally. If you specify both, the image will fix itself to the dimensions you chose.

An example syntax of this is as follows:
<img src=”file”width=”200”>

The HTML style attribute can also be used to add style to an element. This includes color, font, font size, and more. In the case of images, one can specify the height and width of the image using this attribute.

The general syntax is as follows:
<tagname style="property:value;">
