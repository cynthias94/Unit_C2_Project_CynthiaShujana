Character Encoding Schemes

  • ASCII (American Standard Code for Information Interchange) - The first and most widely used character encoding system before Unicode.
  • EBCDIC (Extended Binary Coded Decimal Interchange Code) - Used by IBM mainframes computers until IBM converted to ASCII in the 1980s.
  • ANSI (American National Standards Institute) or Windows-1252- An extended version of ASCII developed by ANSI and used in Windows until Unicode.
  • ISO-8859- Developed by the International organization for Standardization, it was a variety of formats for different languages that existed before Unicode.

Utilizing Unicode

  • Unicode has several character sets. The most popular for web browsers is Unicode Text Format 8-bit, or UTF-8.
  • To force browser compatibility for your web page you use the following code in the head section of your page:
    <meta charset= “utf-8”></li>
  • 38(&), 60(<), 62(>) are some examples of the unicode format of text.

HTML Entities

Several characters are considered reserved for use by HTML. Other characters are difficult to reproduce on the keyboard or are things like emojis that can be produced by the keyboard without a special app. To display these characters as text or emojis, you can write the following code:

  • <li>(nickname method): &+nickname+;
    Example: < becomes & lt ;
  • (number method): & + # + Unicode number +;
    Example: < becomes & # 60 ;

Essential HTML Entities:

Entity Nickname Number
