File Paths

  • A File Path is the route you dictate to the the computer in order to find a file.
  • In the example below: 
            C:\
          Users
           AI
            Documents
              project.docx
    The absolute path of the project.docx file is
    C:\Users\AI\Documents/prpject.docx
    (the absolute path is finding it within the entire system’s constellation of files)
  • If you were working in the AI folder, you could find project.docx’s relative path by telling the computer:
    Documents\project.docx
    (relative path is finding a file in relation to another part of the system)

When to Use Absolute And Relative File Paths?

  • It is best to use absolute paths when referring to files that exist outside of your current computer.
  • It is best to use relative paths when you are referring to files that are on your current computer or server. This way, if the folder that contains the files is moved, the links between files do not break. Users that access the file remotely also will have little problem linking the files.

How Do File Paths Work?

  • Relative path starts at the location of the file we are linking from to another file- it is always relative from linking file’s location.
  • In order to go one level up you would write ../
  • In order to go two levels up from current folder you would write ../../
  • ./ means current folder (used to specify that relative file path and file you want to link to is already is current folder)
  • ./child/ means to link a file that is located in the child folder of the current folder
  • Absolute file path means it only starts in the root file and contains all folders within until it reaches the linked file
  • Don't use the absolute file path which starts from your computer’s root folder
    -If you want to link a static file- download it and specify the relative file path
  • You can use the full url link, but make sure it is accessible
  • If you want to go to a directory on the same level as the current directory, then ./ is not needed
  • Once you get to the same level, you simply go across to the desired folder directory and down to the desired file
