Headings and Paragraphs

Headings

  • Headings are used to create an outline: structure of a web page
  • There are 6 headings types: <h1> through <h6>
  • Aside from the size of the text headings allow assistive devices like screen readers
  • All headings are block level elements
        
      <h1>The top-level heading</h1> 
        <h2>second level</h2> 
          <h3>third level</h3> 
            <h4>fourth level</h4>

The head is where the pageâ€™s metadata is found, along with everything else that is meant not to appear on the visible part of the page.

What should go in the head?

Five types of things are found in the head:

  • Title- the title is what shows up at the top of the web page window or tab.
  • Metadata: data that is invisible to the viewer but is visible to the web crawlers and this important when the page appears in search results.
  • Styles: instructions to the web browser on how the web page should look.
  • Scripts: JavaScript code that must run before the body of the web page loads.
  • Links: important files that are linked to the page, including external CSS and JavaScript files.

Paragraphs

  • Paragraph <p> is the workhouse tag of web design. Most blocks of text use a paragraph for their formatting.
  • Paragraphs are block-level elements, creating space above and below them on the page.
  • If you want to create a line break, use the <br> tag
    <br> will end the text at the point it is inserted and continue it onto the next line. It is an inline element.

Horizontal and Preformatted Text

  • If you want to create a line between blocks of text, use the <hr> element. Hr is an empty tag.
  • If you want to preserve multiple spaces or line breaks for text like in poems, you can use the preformatted text <pre> tag. Pre is a block-level element.

Example:

        <p>poem</p>
        <hr><!-- creates a line- - > 

        <pre> this text in this tag 

              Will display 

            Exactly as 

                 You see it 

        </pre>
