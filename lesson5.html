Iframes

Iframes allow an HTML document to display the contents of one or more other HTML documents. We use the <iframe> element to accomplish this.

  • The src attribute is used to determine the path to the html document or web page that will provide the contents of the iframe.
  • Iframes are inline tags; if the width of an iframe is less than the width of the page, other content may appear to the right or left of it.
  • The title attribute is used to provide screen readers (an in case the source page is inaccessible) a description of the page that should be displayed in the iframe.
  • The basic syntax of iframes is as follows:
    <iframe src=”url” title=”description” width=”” or height=””></iframe>

Styling of Iframes

  • Although not necessary, you can use height and/or width attributes to control the size of the iframe window.
  • You may specify either in terms of pixels (absolute height.width) or percentages (fractions of the current web page window).
  • An example syntax of this is as follows:
    <iframe src=”url” title=”description” width=”400%” height=”300px”></iframe>
