Formatting Tags and Lists

Before beginning, the essential HTML elements must be known. A properly written web page should have the following elements in such order:

  • The HTML's doctype declaration
  • The HTML tag- this begins and ends the page
  • The head tag- this contains the title of the page and other information (the metadata)
  • The title tag- this showcases the text that appears on the browser tab
  • The body tag- this tag contains the visible content showcased on the webpage

The general set-up incorporting these elements:

        <!DOCTYPE html> 
        <html> 
        <head> 
          <title> Title of the page </title> 
        </head> 
        <body> 
        </body> 
        </html>

Types of Tags

HTML has two types of body elements:

  • Block-level elements will automatically appear on separate files on the page.
  • <p> (paragraph tag) and <h1> (heading tag) are examples of block level elements
  • Inline elements can appear next to one another horizontally so that text with an inline element in it will not start a new line.
  • <strong> (turns text bold) and <i> (turns text into italics) are examples of inline elements

Formatting Tags

Several elements modify the look of text within them. These are all the inline elements, you can use them to modify one or more characters within a line of text.

Tags Meaning
<b> </b> This will make text appear bold
<strong> </strong>
<i> </i> This will make text appear in italics
<em> </em>
<mark> </mark> This will make text appear highlighted
<small> </small> This will make text appear smaller
<del> </del> This will make text appear crossed out
<ins> </ins> This will make text appear underlined
<sub> </sub> This will make text appear as a subscript
<sup> </sup> This will make text appear as a superscript

List Elements

There are two types of list elements:

  • <ol> creates an ordered list (a list with numbers)
  • <ul> creates an unordered list (a list with bullets)

Both list types use <li> (list item) for items within a list

  • <ol>, <ul>, and <li> are all block level elements
